- 우드 스틱 1개

- 네일 파일 1개

- 알코올 패드 1개

※ 구성품은 제작 시기에 따라 일부 변경될 수 있습니다.

사용 방법

1. 동봉된 알코올 패드로 손톱 표면의 유분과 이물질을 제거합니다.

2. 손톱 크기에 맞는 네일팁을 선택합니다.

3. 글루 스티커를 부착한 후 네일팁을 올려줍니다.

4. 약 10~15초간 눌러 고정합니다.

5. 제거 시 우드 스틱을 이용하여 가장자리부터 천천히 분리합니다.

사용 시 주의사항

  1. 손톱에 상처, 염증 또는 이상이 있는 경우 사용을 피할 것
  2. 피부가 민감한 경우 사용 전 테스트를 할 것
  3. 무리하게 제거할 경우 손톱 표면이 손상될 수 있으므로 주의할 것
  4. 사용 후 이상 증상이 나타날 경우 즉시 사용을 중단할 것
  5. 보관 및 취급 시 주의사항
    1. 어린이의 손이 닿지 않는 곳에 보관할 것
    2. 직사광선을 피해서 보관할 것
